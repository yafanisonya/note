    1. 样式语法

    1. at语法

    # 注意事项：

    • @charset必须放在第一行
    • at语法必须以分号;结尾
    • 所有符号都是英文符号，区分大小写

    • overflow: hidden; 直接隐藏溢出部分
    • overflow: auto; 灵活设置滚动条

    # 七、盒模型

    • content-box(默认)
    • border-box
      <div class="content-box">content box</div>
  <div class="border-box">border box</div>

    .content-box{
  width: 100px;  
  margin: 25px;
  padding:15px;
  box-sizing: content-box;
  border:5px solid red;
}
.border-box{
  width: 100px;
  margin: 25px;
  padding:15px;
  box-sizing: border-box; 
  border:5px solid green;
}

    • 区别 content-box 的 width 等于 内容宽度 border-box 的 width 等于 内容宽度 + padding + border
